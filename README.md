# The Classical Soch

A premium ASP.NET Core MVC 9 website for a luxury unisex salon and academy in Siwan, Bihar.

## Features
- Premium responsive UI with Bootstrap 5, gold/black/white palette and elegant typography
- Public pages for home, about, services, academy, packages, gallery, FAQ, and contact
- Admin login with session-based authentication
- Clean n-tier structure with controllers, interfaces, services, repositories, DTOs, view models, and shared components
- PostgreSQL-ready ADO.NET repository layer and SQL script folder for stored procedures

## Run locally
1. Install .NET 9 SDK
2. Ensure PostgreSQL is available and update the connection string in appsettings.json
3. Run the SQL scripts in the Database folder
4. Run `dotnet restore`
5. Run `dotnet run`
